# iPhone Evolution / The Glass Timeline

This case study turns a product-history timeline into a museum-tech editorial video.

## What It Proves

- The skill can turn factual milestones into cinematic scenes.
- Important labels remain editable code text.
- Brand-sensitive visuals can be abstracted instead of copied.

## Plugin Route

- Figma: museum storyboard and token board.
- Remotion: data-driven milestone cards from `timeline.story.json`.
- HyperFrames: cinematic opener and closer language.
- Codex: scene assembly, validation, and packaging.
