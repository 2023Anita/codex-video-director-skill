# Image Prompt Patterns

Generate reference images to clarify composition and style. Do not ask the image model to solve editable text, charts, or precise UI logic.

## American SaaS

Use for B2B product videos, launch pages, analytics dashboards, and investor-ready demos.

```text
A premium American SaaS product scene in 16:9.
[Specific UI or workflow subject].
Palette: graphite black, warm white, muted red, steel blue.
Elegant startup launch style, calm, precise, high-end.
No readable text, no logos, no watermark.
```

## Japanese Editorial

Use for anime editorial videos, creator workflows, magazine-style tutorials, and warm educational demos.

```text
A Japanese editorial anime-style scene in 16:9.
[Specific creator/workflow subject].
Palette: warm ivory, sakura pink, ink black, soft cyan.
Airy composition, premium magazine layout, playful but refined.
No readable text, no logos, no watermark.
```

## UI Reference Frame

```text
A clean [light/dark] product interface in 16:9 landscape format.
[Describe layout with left/right/top/bottom placement].
[Describe chart or card shape without relying on exact generated text].
Premium software demo style, precise spacing, strong hierarchy.
No brand logos, no random decorative clutter.
```

## Cutout Character

Use chroma key when a transparent asset is needed.

```text
[Style] illustration of [subject/action].
Palette: [palette].
Flat chroma-key background #00ff00 for background removal.
No text, no UI labels, no watermark.
```

## Background Texture

```text
A 16:9 [style] texture background for an AI workflow video.
[Material, grid, accents, mood].
Minimal, elegant, designed to sit behind animated UI and typography.
No characters, no readable text, no icons, no central object.
```
