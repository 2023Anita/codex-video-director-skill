# Contributing

This project is intentionally small and demo-driven.

Good contributions:

- Add a new visual style reference with a clear storyboard.
- Improve validation without making the skill harder to use.
- Add a focused demo that proves one workflow capability.
- Tighten prompts, templates, or QA checks based on real usage.

Please keep skill instructions concise. Put detailed examples in `references/` or `demo/`.
