# GitHub Project References

These projects shaped the v0.2 direction.

## HyperFrames

- Repository: https://github.com/heygen-com/hyperframes
- Lesson: HTML can be the source of truth for agent-produced video when lint, inspect, preview, and render are part of the workflow.
- Applied here: each case study includes a HyperFrames-ready HTML composition that expresses the cinematic framing and timing language.

## Remotion

- Repository: https://github.com/remotion-dev/remotion
- Lesson: React makes video programmable, reusable, and data-driven.
- Applied here: each case study includes a Remotion-style component skeleton driven by `timeline.story.json`.

## GitHub Unwrapped

- Repository: https://github.com/remotion-dev/github-unwrapped
- Lesson: data becomes memorable when it is shaped as a personal narrative.
- Applied here: product history and technology milestones are treated as narrative datasets, not generic scenes.

## OpenMontage

- Repository: https://github.com/calesthio/OpenMontage
- Lesson: an AI video project feels serious when it presents a complete studio workflow, not one isolated prompt.
- Applied here: v0.2 presents Figma, HyperFrames, Remotion, Codex, sources, validation, and renders as one pipeline.

## Remotion Templates

- Repository: https://github.com/reactvideoeditor/remotion-templates
- Lesson: templates are useful, but shallow without a strong story.
- Applied here: the demos are concrete case studies: iPhone evolution, SpaceX ascent, and AI interface futures.
