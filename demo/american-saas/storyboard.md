# American SaaS Demo Storyboard

Aspect ratio: 16:9  
Target duration: 36 seconds  
Audience: founders, indie hackers, AI builders, product marketers  
Style direction: polished American SaaS launch video, graphite UI, restrained red accents

## Scene 1: Hook

Goal: Show that the skill turns unclear video intent into a coded production pipeline.

Visual reference: `assets/american-hero-cover.png`

Exact text:
- Codex Video Director
- From storyboard to coded product video

Motion:
1. Dark workspace fades in.
2. Three pipeline cards slide up: Storyboard, Image Reference, Coded Motion.
3. Red accent line connects the cards.

Acceptance checks:
- Title readable in first 2 seconds.
- Pipeline cards are visually distinct.
- No generated image text is needed.

## Scene 2: Product Proof

Goal: Show a coded analytics scene that looks like a real B2B product video.

Visual reference: `assets/american-analytics-ui.png`

Exact text:
- 68%
- already solved by docs
- 4.2 hrs saved weekly

Motion:
1. Dashboard frame slides in from the right.
2. Donut chart draws clockwise.
3. Metric card pops in after the chart.

Acceptance checks:
- Donut chart is code-rendered.
- Metrics are editable text.
- Motion feels calm, not flashy.

## Scene 3: Human Layer

Goal: Show the founder/creator layer without turning the video into generic stock footage.

Visual reference: `assets/american-founder-cutout.png`

Exact text:
- Decide the visual first
- Rebuild the scene in code

Motion:
1. Character cutout enters from bottom left.
2. Two annotation labels fade in.
3. UI cards dim slightly to create depth.

Acceptance checks:
- Character does not cover the metrics.
- Labels fit inside safe margins.
- Foreground/background hierarchy is clear.

## Scene 4: Close

Goal: Convert interest into GitHub installation intent.

Visual reference: generated from scene assets

Exact text:
- Install the skill
- Ship coded motion demos
- github.com/anita/codex-video-director-skill

Motion:
1. Demo layers collapse into a clean package card.
2. `.skill` badge appears.
3. GitHub URL fades in.

Acceptance checks:
- Close screen works as a README thumbnail.
- `.skill` package is the hero object.
- GitHub URL remains legible.
